# 알고리즘 계획
## 1. 메인화면 알고리즘
* ### 게임시작 번호 (1번) 입력시 게임 시작
* ### 불러오기 번호 (2번) 입력시 게임 불러옴
* ### 게임종료 번호 (0번) 입력시 게임 종료

## 2. 캐릭터 생성 알고리즘
* ### 1. 닉네임 입력
* ### 2. 입력 글자 중 소문자를 대문자로 변환
* ### 3. 입력 글자의 길이가 1~12 글자 범위가 아닐 경우 오류문 출력 후 1번으로 돌아감.
* ### 4. 입력 글자 중 영어를 제외한 문자가 있을 시 오류문 출력 후 1번으로 돌아감.
* ### 5. 닉네임 저장
